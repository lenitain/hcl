# SetFuncs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** Implement 6 set-theory functions (setintersection, setunion, setsubtract, setsymmetricdifference, setproduct, setissubset) for HCL's built-in function library.

**Architecture:** HCL represents sets as arrays with unique elements. Follow existing `fn_distinct` pattern using `to_json_string()` as dedup key. Register functions via `FuncDefBuilder` in `builtin_functions()`.

**Tech Stack:** MoonBit, existing `FuncDef`/`FuncDefBuilder` pattern, `Set::new()` for dedup.
